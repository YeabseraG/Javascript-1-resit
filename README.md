# GameHub

A modern online game store built with HTML, CSS and JavaScript using the Noroff GameHub API.

## Description

GameHub is an online game store that allows users to browse games, view detailed product information, add products to a shopping cart, and complete a checkout process.

The application retrieves all product information dynamically from the GameHub API and stores cart data in localStorage.

## Features

* Browse games from the GameHub API
* Filter games by genre
* View individual product pages
* Add products to cart
* Remove products from cart
* View checkout summary
* Complete checkout process
* Order confirmation page
* Responsive design for desktop and mobile devices
* Loading indicators
* Error handling
* Toast notifications for cart actions

## Built With

* HTML5
* CSS3
* JavaScript (ES6 Modules)
* GameHub API

## Getting Started

### Installing

1. Clone the repository

```bash
git clone https://github.com/YeabseraG/Javascript-1-resit.git
```

2. Open the project in Visual Studio Code

3. Run the project using Live Server

### Running

Open `index.html` with Live Server or another local development server.
